# Burger Sipariş Sitesi - MVC

Bu MVC projesi, hızlı ve güvenilir bir burger sipariş deneyimi sunmak amacıyla müşteri arayüzünden başlayarak site yöneticisinin admin paneline kadar geniş bir işlevselliği kapsamaktadır. Kullanıcılar onlara sunulan burger ve diğer ürünleri sipariş edebilir, admin ile iletişime geçebilir, önceki siparişlerini görüntüleyebilir. Admin ise tüm bu süreçleri yönetecek çok detaylı bir panel kullanımına sahiptir.
